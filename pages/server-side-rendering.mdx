# Серверный рендеринг

## Не поддерживается

Inertia в настоящее время не поддерживает рендеринг на стороне сервера. Если это приоритет для Вашего приложения, Inertia может быть неправильным выбором для Вас. Тем не менее, мы считаем, что рендеринг на стороне сервера технически возможен с помощью Inertia, и, вероятно, вернемся к этому в будущем.

## Сервисы предварительного рендеринга

## Laravel

**[Depictr](https://github.com/juhlinus/depictr)** - пакет, разработанный [Linus Juhlin](https://twitter.com/juhlinus), который отображает статические страницы для поисковых роботов.

## Javascript/Node.js

Некоторые пользователи Inertia успешно используют такие службы, как [Prerender.io](https://prerender.io), [Prerender.cloud](https://www.prerender.cloud) и [Rendertron](https://github.com/GoogleChrome/rendertron) для создания и кеширования HTML-версий своих веб-сайтов Inertia, а затем для передачи этого контента ботам в целях SEO.
